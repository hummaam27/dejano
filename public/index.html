déjàno.

a daily AI generator that refuses to repeat itself.

Type any topic. Click Generate repeatedly. déjano remembers what it already wrote and refuses to repeat itself — if the model drifts toward something semantically close to a past answer, it gets rejected and retried.

Advanced — use a curated topic pack instead

Topic packs are YAML files in topics/ with curated subtopic seeds. Useful for production use where you want consistent style and forced topic rotation.

How to use: Type a topic, hit Generate. The first answer is always fresh. Click again — the model has to find a new angle. After 5–8 clicks, you'll start seeing rejection cards: déjano caught the model trying to repeat itself. Use Burst ×10 to fast-forward to that moment.
